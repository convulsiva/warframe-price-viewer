# Trading Console Redesign / Новая торговая консоль

## English

- Completely redesigned desktop interface with improved window resizing.
- Added English and Russian languages with bilingual item search.
- Added saved-item management, seller reputation and local price history charts.
- Improved navigation, filters, themes and visual clarity.

## Русский

- Полностью обновлён интерфейс и улучшена адаптация под размер окна.
- Добавлены русский и английский языки с двуязычным поиском предметов.
- Добавлены удобное управление избранным, репутация продавцов и локальные графики истории цен.
- Улучшены навигация, фильтры, темы и читаемость интерфейса.
